<br />
"MySQL" database is used in Project
<br />
Features of Project:
<br />
1} Add a book
<br />
2} Delete a Book
<br />
3} Allocate a Book to Student
<br />
4} Deallocate a Book from Student
<br />
5} View Availability of Books in Database
